📦 Installatie

1

Plugin Uploaden

Upload de plugin bestanden naar /wp-content/plugins/ en activeer in WordPress Admin

2

Supabase Configureren

Voer je Supabase URL en API key in via Settings → Barbershop Booking

3

Database Setup

Zorg dat je Supabase database de juiste tabellen heeft (services, barbers, appointments)

4

Widget Plaatsen

Sleep de widget naar je gewenste widget area of gebruik de shortcode [barbershop_booking]
